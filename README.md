# Thumblify - AI Thumbnail Generator

A full-stack AI-powered YouTube thumbnail generator built with React, Node.js, Express, MongoDB, and Pollinations AI. Users can generate professional, eye-catching thumbnails by simply entering their video title, selecting a style, color scheme, and aspect ratio.

---

## 🚀 Features

- AI-powered thumbnail generation using Pollinations AI
- Multiple thumbnail styles — Bold & Graphic, Tech/Futuristic, Minimalist, Photorealistic, Illustrated
- Multiple color schemes — Vibrant, Sunset, Forest, Neon, Purple, Monochrome, Ocean, Pastel
- User authentication with session-based login/signup
- Thumbnail history saved per user
- Cloud image storage using Cloudinary
- Responsive and modern UI built with React

- React + TypeScript

- Node.js + Express
